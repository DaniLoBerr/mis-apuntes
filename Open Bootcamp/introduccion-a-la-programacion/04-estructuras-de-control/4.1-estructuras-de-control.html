Estructuras/Sentencias de control

Serie de órdenes/instrucciones que permiten modificar el flujo de ejecución de un programa. Pueden ser secuenciales, iterativas o de control avanzadas. Se basan en la utilización de condiciones para comparar situaciones y que el programa ejecute una u otra opción según la situación. Cuando se evalua una condición, solo se pueden dar dos resultados (lógica booleana). Con las estructuras de control se puede: - De acuerdo con una condición, ejecutar un grupo u otro de sentencias (If-Then-Else) - De acuerdo con el valor de una variable, ejecutar un grupo u otro de sentencias (Switch-Case) - Ejecutar un grupo de sentencias solo cuando se cumpla una condición (Do-While) - Ejecutar un grupo de sentencias hasta que se cumpla una condición (Do-Until) - Ejecutar un grupo de sentencias un número determinado de veces (For-Next)

Tipos de condiciones

  • Condiciones lógicas
    Y
    Comparación en la que se deben cumplir dos o más situaciones para que las ordenes se ejecuten. Habitualmente, en informática, se representa como: "Y" = &&
    O
    Comparación en la que se debe cumplir o una u otra o ninguna de las situaciones para que las ordenes se ejecuten. Habitualmente, en informática, se representa como: "O" = ||
  • Condiciones comparativas
    • Mayor que (>)
    • Menor que (<)
    • Mayor o igual que (>=)
    • Menor o igual que (<=)
    • Igual a (==)
    • Distinto a (!=)

Estructura de control de selección

Condicional simple: if / else if / else

Si se cumple la condición o las condiciones (if), se ejecutan todas las acciones indicadas en el cuerpo de la sentencia. En caso contrario, si se cumple otra condición (else if), se ejecutan las acciones contenidas en esa sentencia. En cualquier otro caso (else), si no se ha cumplido ninguna de las condiciones anteriores, se ejecutarán las de esta última sentencia, denominada fallback. El fallback es la acción última por defecto, la acción que se tiene que hacer porque, por circunstancias del código, se debe realizar alguna acción.

No es necesario escribir toda la sintaxis completa, se puede utilizar solo un if por ejemplo. Lo importante es saber que, a nivel lógico, se puede tener una condición, y esa condición se puede evaluar de forma consecutiva: si pasa esto, haz esto; si pasa esto otro, haz esto otro; y, en cualquier otro caso, haz esto último.

if es una sentencia imperativa

Los if están en casi todos los lenguajes de programación.
