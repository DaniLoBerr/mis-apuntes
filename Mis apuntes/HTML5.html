¿Qué es HTML?

 HyperText Markup Language= Lenguaje de marcado de hipertexto. Formado por etiquetas. Una etiqueta es un elemento que contiene información para darle un significado determinado. Forman la base del documento HTML. Existe un número finito de etiquetas, son elementos concretos, y cada uno tiene una función específica. Se pueden anidar etiquetas dentro de otras etiquetas.

 Acceso a una página web:

  1. El navegador solicita un documento de texto (documento HTML) al servidor.
  2. El navegador muestra el contenido interpretado.

Semántica en HTML5

 En los documentos HTML se estructura y se dota de significado a la información. Especialmente en HTML5, las etiquetas se utilizan para indicar información semántica de la página web, dejando todo lo relacionado con el aspecto visual al lenguaje de estilos CSS (separación de la presentación al contenido). El objetivo de esto se debe a la tendencia a un internet autónomo capaz de procesar, analizar y entender la información para un mejor posicionamiento en la web.

Estructura de una etiqueta HTML

< etiqueta atributo="valor" > Contenido < /etiqueta >

Etiqueta
 Es posible escribirla en mayúculas pero se recomienda siempre hacerlo en minúsculas. No se puede escribir cualquier cosa en forma de etiqueta (elementos concretos). Para la mayoría es obligatorio escribir una etiqueta de cierre, aunque existen excepciones.
Atributo
 Determinan cierta información sobre la etiqueta en la que se encuentran o su modo de actuar. Cada una tiene una misión y comportamiento concreto. La mayoría van asociados a un valor. Existen atributos que son específicos de ciertas etiquetas (y que pueden ser opcionales u obligatorios) y otros que son válidos para varias etiquetas.
 Es posible escribir el valor de un atributo entre comillas simples pero se recomienda hacerlo entre comillas dobles.
 Existen 3 tipos de atributos:
  • Conjunto finito de valores. Aquellos que tienen un número limitado de valores válidos.
  • Valores libres. Como una URL o un determinado texto.
  • Valores booleanos. Aquellos que solo tienen como valor true o false, aunque no es válido indicar literalmente su valor: Para indicar que el valor es true, se puede escribir el atributo sin más, se puede escribir atributo="" o se puede poner como valor el nombre del atributo. Para indicar que el valor es false, directamente no se escribe el atributo (es el valor por defecto).
